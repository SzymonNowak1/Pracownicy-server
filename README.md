# WORKERS

## Przygotowanie
* Baza danych PostgreSQL: skrypty inicjujące z *src/main/resources/db*
* Namiary na bazę ustawione w *src/main/resources/application.properties*

## Uruchomienie
* Uruchom *src/main/java/pl/szynow/workers/WorkersApplication*

* Wykonaj polecenia:


    cd client
    npm install
    ng serve
